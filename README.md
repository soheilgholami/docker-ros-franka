# docker-ros-franka

A Dockerfile that sets up the following dependencies: 
- [ROS Noetic Desktop-Full](https://wiki.ros.org/noetic/Installation/Ubuntu)
- [libfranka (0.8.0)](https://github.com/frankaemika/libfranka)
- [franka_ros](https://github.com/frankaemika/franka_ros)
- [Pinocchio](https://github.com/stack-of-tasks/pinocchio)
- [Ruckig](https://github.com/pantor/ruckig)
- [MatLogger2](https://github.com/ADVRHumanoids/MatLogger2)
